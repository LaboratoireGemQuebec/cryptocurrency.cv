# Layer 2 / Scaling Data APIs

> Layer 2 rollup TVL, transaction activity, and scaling solution analytics.

---

## L2BEAT

| | |
|---|---|
| **Base URL** | `https://l2beat.com/api` |
| **Key Required** | No |
| **Rate Limit** | 30/min |

**Used Endpoints:**

| Endpoint | Purpose |
|---|---|
| `GET /scaling/tvl` | L2 TVL data |
| `GET /scaling/activity` | L2 activity metrics |
| `GET /activity` | Transaction activity |
