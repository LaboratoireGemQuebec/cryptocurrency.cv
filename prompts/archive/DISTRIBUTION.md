# Distribution & Submission Prompts

Actionable prompts designed to be pasted into new Claude Opus 4.6 chat sessions. Each prompt is self-contained with full context so the agent can execute independently.

## How to Use

1. Open a new Claude chat (Opus 4.6)
2. Copy the **entire contents** of one prompt file
3. Paste it in and let the agent execute
4. Review outputs, submit where manual steps are needed
